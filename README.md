# ATM Simulator with Tkinter

A Python-based ATM Simulator with a graphical user interface using Tkinter. The app allows users to:
- Check balance
- Deposit money
- Withdraw money
- Transfer funds
- View transaction history

## Requirements
- Python with the `ttkthemes` library installed.
